## 케이스 주제

Q. react의 useState함수를 이해하기 위해 구현을 해보자.


## 기능 요구사항

1. user action을 정의한다.


2. action에 따른 state를 변경한다.


3. state 변경 시 virtual dom tree를 갱신한다.


4. 갱신 된 virtual dom tree로 html node로 생성하여 반영한다.
